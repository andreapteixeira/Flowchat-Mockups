Social Platform Guidelines

Main Social Platforms Daily Limits

Use caution when interacting with users in the major social platforms. These limit indicators can preven t you account from being blocked.

Facebook: Don’t exceed the 30 friend requests and sending initial messages per day.

Instagram: Don’t exceed the 50 follower requests and sending initial messages per day.

Linkedin: Don’t exceed the 25 connections requests and sending initial messages per day.

Twitter: Don’t exceed the 400 profile follows and sending more than 1000 messages per day.

What do Messenger Icons mean?

Status: Sending...
This icon shows for about 1 second when you send a message and changes to the “sent” icon. If it stays in this stage for more than 3 seconds, FB might be starting to throttle your messages, time to switch to only doing replies to existing conversations to prevent block.

Status: Sent
Your message has been sent to the recipent.

Status: Delivered
Your message has been delivered to the recipent, but he didin't read the message.

Status: Read
If Your message has been read by the recipent,the small profile picture will show.

Status: Error / Temporarily can't send messages to new conversations
At this stage you can only reply to messages from existing conversations. Avoid getting to this stage as much as possible.

Instagram Pro Tip

The best process for unfollowing on IG after following someone is to send them a message. Unfollowing accounts fall in the same limit bucket as follows. If someone does unfollow, they'll end up not being able to do more follows ( limit is about 50 a day). Example: If you do, 30 unfollows, you can only do 20 follows/ 20 more unfollows until you will hit the limit.
